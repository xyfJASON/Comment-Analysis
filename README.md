# 大一立项——结合用户评价分析的网店假货预警

任何一个评论都可以分为 5 类：与商品质量无关、是水军、情感正向、情感中性、情感负向。于是，我们只需将一个评论分为这 5 类之一，这就是一个**文本分类**问题。
